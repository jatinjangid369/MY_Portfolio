# Jatin's Portfolio ⚡

![Portfolio Preview](https://via.placeholder.com/800x400?text=Your+Portfolio+Screenshot)

A clean, beautiful, and fully responsive **React** portfolio website built to showcase my skills, projects, experience, and achievements.

## ✨ Features

- Modern and responsive design (mobile-friendly)
- Smooth scroll animations (using React Reveal)
- Lottie animations for engaging visuals
- Dynamic color extraction from images
- Sticky header with hide-on-scroll effect
- Twitter feed embedding
- Easy to customize

## 🛠️ Tech Stack

- **Frontend**: React (v16) + Create React App
- **Styling**: Sass (SCSS)
- **Animations**: React Reveal, Lottie React
- **Utilities**: ColorThief, React Headroom, React Twitter Embed, React Easy Emoji
- **Others**: Prettier, Jest & Enzyme (for testing)

## 📂 Project Structure (Important Folders)

- `src/` — Main source code
- `src/portfolio.js` — Edit this file to change all your personal information
- `src/_globalColor.scss` — Change global theme colors
- `public/` — Static assets and index.html
- `build/` — Generated folder after running `npm run build`
